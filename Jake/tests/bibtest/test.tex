        Nun, ein \say{kommutativer Ring mit Einselement, indem für jedes Element genau ein multiplikatives Inverses existert}. 
        Einfach, nicht? Jetzt fehlt ja nur noch, was ein Ring genau ist... Nun, es freut mich dass du fragst: \say{Ein Ring ist eine \jmark[Menge]{mrk:Menge} auf der zwei Verknüpfungen definiert sind (\(+,\,\cdot: \R \times \R \rightarrow \r\)), wobei erstere (\(+\)) Addition genannt wird und sowohl assoziativ, als auch kommutativ zu sein hat als auch ein neutrales Element besitzen muss, welches wir als Null Element bezeichnen.
        Weiter muss es noch ein additives Inverses geben, welches für ein \(a\) mit \(-a\) bezeichnet wird und für das gilt: \(a+(-a) = 0\). 
        Zweitere Operation \(\cdot\) nennen wir Multiplikation und fordern von ihr assoziativ eigenschaften aufzuweisen (ist sie zusätzlich kommutativ so bezeichnen wir sie als kommutativen Ring). 
        Für beide Operationen kombiniert muss zudem das Distributiv gelten, damit wir das Mengenkonstrukt auch wirklich als Ring bezeichnen}. \newline
